DriveX - Premium Used Car Marketplace

Project Description

DriveX is a responsive web-based vehicle marketplace developed using HTML, CSS, and JavaScript. The platform allows users to browse, buy, and sell premium used vehicles through an easy-to-use interface.

Features

Home Page with featured vehicles

Vehicle Listings

Vehicle Details Page

Buy and Sell Vehicle Options

User Registration and Login

User Profile Management

Wishlist Functionality

Secure Payment Page

Contact Page

Admin Dashboard

Responsive Design


Technologies Used

HTML5

CSS3

JavaScript

Font Awesome Icons
